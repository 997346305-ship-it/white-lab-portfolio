## 部署到 Vercel

这是当前这个作品集最适合的上线方式，够轻，也适合面试时直接发链接。

### 上线前你只需要确认

- 不想展示的项目已经隐藏
- 本地预览没有明显错位
- 新补的图片和视频路径都能正常打开
- 如果后面继续补大视频，优先压成网页展示版再上传

### 部署步骤

1. 把整个项目上传到 GitHub
2. 打开 [Vercel](https://vercel.com/)
3. 用 GitHub 账号登录
4. 点击 `Add New`
5. 选择 `Project`
6. 选中这个仓库并导入
7. 保持默认设置直接部署

这个站是纯静态站，不需要额外的构建命令。

### 部署时建议

- Root Directory 选项目根目录
- Build Command 留空
- Output Directory 留空
- Framework Preset 选 `Other`

### 预览检查

部署完先检查这几项：

- 首页是否正常进入
- 品牌设计页是否能正常打开详情
- 黑底查看页是否能 `Esc` 退出
- 左右方向键是否能切换图片
- 视频项目是否能正常播放
- 小红书图标跳转是否正常

### 如果后面要正式给面试官看

建议顺序：

1. 先发 Vercel 预览链接给自己或朋友看一遍
2. 确认手机和电脑都能打开
3. 再决定是否绑定自定义域名

### 当前项目里已经帮你准备好的内容

- `vercel.json`：静态资源缓存策略
- `.vercelignore`：避免把备份、临时文件和工具脚本一起传上去

### 现在不急着做的事

- 不必马上买域名
- 不必现在就做正式备案类操作
- 不必为了上线删除本地原始素材
