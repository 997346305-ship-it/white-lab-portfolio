## 本地网页的稳定启动方式

`http://127.0.0.1:8000/` 是电脑本机地址，必须有本地服务运行才能访问。

现在已经改为 Windows 启动文件夹和隐藏守护进程托管：

- 每次登录 Windows 后自动启动
- 服务异常退出时会在几秒内自动恢复
- 重复点击启动文件不会重复占用端口
- 不依赖一直开着的命令行窗口

## 日常使用

双击 `启动网页.bat`，它会检查服务状态，必要时自动恢复，然后打开首页。

只想恢复服务、不打开浏览器时，双击 `只启动服务.bat`。

## 重新安装自动启动

双击 `安装开机自启.bat` 后，会在当前用户的 Windows 启动文件夹中创建：

`White Lab Local Portfolio.lnk`

它不需要管理员权限。每次登录 Windows 时，守护进程会在后台自动启动。
